Archery for beginners

In good archery, the archer first shoots and then XXX. We can represent an Archer as follows:

Archer:{
  .heading: Direction;
  XXXXXXXXXX
  }

A XXXXXX reader may have noticed that we are declaring .heading before .aiming, even if the archer is supposed to shoot (.aiming) and then move (.heading). However, this is fine: The order of the declarations is irrelevant.

We can then write specific kinds of Archers. For example, this is an Archer that will head North after shooting East.

ArcherNE:{
  .heading -> North; .aiming -> XXXXXXX;
  }
